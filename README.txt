Ajedrez en Python
Este es un programa simple de ajedrez en Python. El tablero se representa como una matriz de 8x8, 
y cada pieza está representada por su símbolo en Unicode. El programa permite una partida entre dos 
jugadores, donde cada jugador puede mover sus piezas de acuerdo a las reglas del ajedrez.

Características Implementadas
  Tablero de ajedrez representado como una matriz.
  Movimiento de piezas según las reglas del ajedrez.
  Interacción a través de la consola.

Funcionalidades Pendientes
Aún falta implementar las siguientes características:
  Jaque.
  Enroque.
  Ahogado.
  Evitar que las piezas que no sean caballos puedan saltar sobre otras piezas.

Interfaz
La partida se juega desde la consola, ya que actualmente no cuenta con interfaz gráfica.

English 
Chess in Python
This is a simple chess program in Python. The board is represented as an 8x8 matrix, 
each piece is represented by its Unicode symbol. The program allows a game between 
two players, where each player can move their pieces according to the rules of chess.

Implemented Features
  Chessboard represented as a matrix.
  Piece movement according to the rules of chess.
  Interaction via console.

Pending Features
The following features are still pending implementation:
  Checkmate.
  Castling.
  Stalemate.
  Preventing non-knight pieces from jumping over other pieces.

Interface
The game is played from the console, as it currently does not have a GUI.
